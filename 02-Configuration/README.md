# 02 - Configuration

## 1. Overview

This section outlines the configuration of Microsoft Purview Communication Compliance used within this lab.

---

## 2. Roles and Permissions

The following roles were relevant to configuration and management activities:

- Compliance Administrator
- Communication Compliance Administrator
- Communication Compliance Analyst

---

## 3. Policy Setup

A Communication Compliance policy was configured to monitor in-scope user communications across supported channels.

### Key Configuration Areas

- Policy scope
- Communication channels
- Conditions and indicators
- Reviewer assignment

---

## 4. Configuration Approach

The configuration was designed for a controlled test scenario using pilot users to avoid unintended impact on wider users.
