# 01 - Overview

## 1. Overview

This section provides the purpose, scope, and structure of the Purview Communication Compliance lab.

---

## 2. Lab Objective

The objective of this lab is to simulate the setup and validation of Microsoft Purview Communication Compliance controls in a controlled test environment.

---

## 3. Scope

This repository covers:

- Communication Compliance policy setup
- Controlled testing activities
- Alert review and investigation

---

## 4. Expected Outcome

The expected outcome is a clear demonstration of how Communication Compliance can be configured, tested, and investigated in a structured and professional manner.
